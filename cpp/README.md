# C++ Black-Scholes Greeks & Pricing Calculator

## How to Build & Run

1. Make sure you have a C++ compiler (e.g., g++):
   ```sh
   g++ --version
   ```
2. Compile and run:
   ```sh
   g++ -std=c++11 -o bsm_greeks bsm_greeks.cpp
   ./bsm_greeks
   ```
