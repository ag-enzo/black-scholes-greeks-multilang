# TypeScript/Node.js Black-Scholes Greeks & Pricing Calculator

## How to Run

1. Make sure you have Node.js and npm installed:
   ```sh
   node --version
   npm --version
   ```
2. Install dependencies and compile:
   ```sh
   npm install
   npx tsc bsm_greeks.ts
   ```
3. Run the program:
   ```sh
   node bsm_greeks.js
   ```
